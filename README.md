# Helping Hands App – Authentication Screens

## 📱 Project Overview
Helping Hands is an NGO-focused mobile application designed to connect donors, volunteers, and people in need.

This repository showcases the **authentication UI screens** designed as part of a UI/UX project.

---

## 🎯 Objective
To design a **simple, friendly, and trustworthy** authentication flow for users accessing the Helping Hands app.

---

## 🧭 Screens Included

### 1. Splash Screen
- Displays Helping Hands logo
- Reinforces brand identity
- Calm and welcoming color palette

### 2. Welcome Screen
- Primary actions:
  - Log In
  - Sign In
- Clear call-to-action buttons

### 3. Role Selection Screen
- User roles:
  - Donor
  - Volunteer
  - Receiver
- Enables role-based experience

### 4. Login Screen
- Email / Phone number input
- Password field
- Forgot Password option
- Primary login CTA

### 5. Sign-Up Screen
- Email / Phone number
- Password & confirm password
- Simple and minimal form design

---

## 🎨 Design Style
- Soft purple background for trust & calmness
- Rounded buttons for approachability
- Minimal layout to reduce cognitive load

---
